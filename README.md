# Project Setup

### Steps to Run Locally

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Set Up Environment Variables**  
   Create a `.env` file in the root directory with the following variables:

   ```plaintext
   MONGO_URI=
   JWT_SECRET=shu-project
   TWILIO_SID=
   TWILIO_AUTH_TOKEN=
   PORT=5000
   ```

3. **Start the Development Server**
   ```bash
   npm run dev
   ```
